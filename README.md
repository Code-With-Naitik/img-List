# AI Fashion Studio — Premium Client Experience

The luxury front-end client interface for **AI Fashion Studio**, an Apple-inspired MERN startup helping saree shops and ethnic fashion brands generate photorealistic model lookbooks.

---

## ✨ Features
- **Luxury Fashion Aesthetic**: Black-and-gold color palettes, premium serif typography, elegant glassmorphism effects.
- **Dynamic AI Free Sample Intake**: Upload saree flat-lays and dispatch GPU queue requests.
- **Interactive Shoot Configurator**: Live pricing calculators, location scouts, and model archetype setups.
- **High-Fidelity Admin Dashboard**: Full CRUD console featuring dynamic GPU pipeline tracking and real-time database controls.

---

## 🚀 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Launch Local Dev Client
```bash
npm run dev
```
The client will launch ready on `http://localhost:5173/` or the next active port.
